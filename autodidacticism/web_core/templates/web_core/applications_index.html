Finance Applications

Lazy Prices Algorithm Data Pipeline:

Lazy Prices refers to a research paper published by Lauren Cohen, Christopher Malloy and Quoc Nguyen that explores the relationship between the changes to the language and construction of a company’s financial reports and future financial performance. As a result of the findings described in the paper, many quantitative analysts incorporate data about a firm’s quarterly and annual reports into their trading strategies. The applications/packages below are all elements that are used to create an automated pipeline for the extraction and transformation of financial reporting data, as well as ingestion of said data into a trading algorithm back-testing application or similar project. More information on the Lazy Prices research paper and development of the Lazy Prices data pipeline can be found in the Resources section here.
