Hydrodynamic Modelling

Hydrodynamic modelling is the use of computational models to simulate currents, water levels, salinity etc. DHI Water & Environment provides complex modeling software in the form of MIKE 21 and MIKE 3. These data models perform calculations based on boundary conditions and GIS mesh data. Data output from MIKE HD models is output in the DFS (Data File System) format. This Binary data file format provides unique challenges when creating in-house data pipelines.

Hydrodynamic Model Data File System Data Pipelines

Several projects for clients required data generated from the Hydrodynamic models to be imported into various other applications such as a web data dashboard or external database. This required several creative in-house workarounds due to the unique nature of the DFS file format and its interaction with other applications and operating systems. The goals of creating a more permanent solution to transferring data between processes lead to the dfs data pipeline library. Most of the library is proprietary as it deals with client and organization specific processes/pipelines however some of the more generalized and simplistic elements (such as the Mail Socket API) are publicly available.

Finance Applications

Lazy Prices Algorithm Data Pipeline:

Lazy Prices refers to a research paper published by Lauren Cohen, Christopher Malloy and Quoc Nguyen that explores the relationship between the changes to the language and construction of a company’s financial reports and future financial performance. As a result of the findings described in the paper, many quantitative analysts incorporate data about a firm’s quarterly and annual reports into their trading strategies. The applications/packages below are all elements that are used to create an automated pipeline for the extraction and transformation of financial reporting data, as well as ingestion of said data into a trading algorithm back-testing application or similar project. More information on the Lazy Prices research paper and development of the Lazy Prices data pipeline can be found in the Resources section here.
